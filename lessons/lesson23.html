Lesson 23 — Subscript <sub>

← Back to Curriculum ← Previous Lesson Next Lesson →
Estimated time: 25–35 minutes · Learn how to use <sub> for chemical formulas, math, and references

Learning Goals

  • Understand the semantic role of <sub>.
  • Use <sub> correctly in math, chemistry, and notation.
  • Learn accessibility concerns when using subscript.
  • Avoid misusing <sub> for visual layout.

Part 1 — What is <sub>?

The <sub> element is used to display subscript text — text that appears slightly lower and smaller than the baseline. It’s common in scientific formulas, math notation, and chemical symbols.

Part 2 — Example

<p>Water formula: H<sub>2</sub>O</p>
<p>Math index: a<sub>n</sub> = a<sub>n-1</sub> + d</p>

Result: The “2” in H2O is lowered, as are subscripts in the math equation.

Part 3 — Best Practices

  • Use for **scientific or technical notation** only.
  • For styling text smaller/lower, use CSS instead.
  • Combine with <sup> (superscript) when writing mathematical series or chemical equations.

Part 4 — Accessibility Notes

Screen readers read <sub> text as normal, without announcing “subscript.” To clarify meaning, provide surrounding context: e.g., “the chemical formula for water, H2O.”

Part 5 — Common Mistakes

  • ❌ Using <sub> to make disclaimers or footnotes → use <small> instead.
  • ❌ Overusing subscripts where plain text would suffice.
  • ❌ Expecting browsers to style it identically everywhere (font differences exist).

Quick Quiz

  1. What is the purpose of <sub>?
  2. Give an example use case in chemistry.
  3. Why should you avoid <sub> for disclaimers?
Sample Answers
  • To display text as subscript (lowered below baseline).
  • In H2O, the 2 is a subscript.
  • Because <small> is semantically correct for disclaimers.

Lesson 23 Dictionary

terms
<sub> (Subscript)
Displays text slightly below the baseline (e.g., H2O).
Baseline
The invisible line on which most characters sit. Subscript text drops below it.
