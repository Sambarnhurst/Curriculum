Part 1: What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It structures the content of the page: text, images, links, and layout.

<!DOCTYPE html>
<html>
  <body>
    Hello World!
  </body>
</html>

Practice: Write down three things you already knew (or just learned) about HTML.

Part 2: HTML vs CSS vs JavaScript

  • HTML: Structure (the skeleton)
  • CSS: Style (the clothing)
  • JavaScript: Behavior (the brain)

Together they make the modern web.

Part 3: Browsers & Rendering

Browsers (Chrome, Firefox, Safari) read HTML and render it visually.

They parse the HTML into a DOM (Document Object Model).

Part 4: Writing Your First HTML File

Every HTML file ends with .html. Example:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    Hello, World!
  </body>
</html>

Part 5: The Doctype & Root Element

<!DOCTYPE html> tells the browser: “This is HTML5.”

<html> wraps the entire document.

Part 6: The Head Section

The <head> contains metadata, links, scripts, and the title.

<head>
  <title>My First Page</title>
  <meta charset="UTF-8">
</head>

Part 7: The Body Section

Visible content (text, images, links, etc.) goes inside <body>.

<body>
  <h1>Welcome to my page</h1>
  <p>This is my first paragraph.</p>
</body>

Part 8: Headings & Paragraphs

HTML offers 6 levels of headings:

<h1>Main Title</h1>
<h2>Subtitle</h2>
<h3>Sub-subtitle</h3>

Paragraphs are made with <p>.

Part 9: Comments & Clean Code

Comments help explain code:

<!-- This is a comment -->

Always indent nested elements for readability.

Part 10: Mini Project – Personal Bio Page

Try building a simple bio page:

<!DOCTYPE html>
<html>
  <head>
    <title>My Bio</title>
  </head>
  <body>
    <h1>Your Name</h1>
    <p>A short paragraph about you.</p>
    <!-- Add a fun fact -->
  </body>
</html>

Challenge: Add headings, paragraphs, and a comment. Save it as bio.html and open it in your browser.
