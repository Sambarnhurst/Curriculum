Lesson 2 ‚Äî HTML vs CSS vs JavaScript

HTML defines structure & meaning, CSS defines presentation, and JavaScript adds behavior. Keeping them separate improves clarity and maintainability.

Learning Goals

  • Understand the role of each layer: HTML, CSS, and JavaScript.
  • Explain why we separate structure, style, and behavior.
  • Recognize basic syntax differences.

1 ‚Äî HTML: Structure

HTML provides the skeleton of a webpage ‚Äî the content and semantics.

<h1>My Website</h1>
<p>Welcome to my site.</p>
<a href="about.html">About me</a>
<img src="photo.jpg" alt="A photo">

2 ‚Äî CSS: Style

CSS handles presentation ‚Äî colors, spacing, fonts, layout.

h1 { color: cyan; font-size: 2rem; }
p  { color: #b6b6b6; }
img{ border-radius: 8px; }

3 ‚Äî JavaScript: Behavior

JavaScript reacts to user actions and updates the page.

document.getElementById("clickMe")
  .addEventListener("click", () => {
    alert("You clicked the button!");
  });

Example HTML element expected: <button id="clickMe">Click me</button>

Putting It All Together

This assembles the three snippets into a single runnable page:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Lesson 2 Demo</title>
  <style>
    body { background:#121212; color:#eee; font-family: system-ui, sans-serif; }
    h1 { color: cyan; }
  </style>
</head>
<body>
  <h1>Hello!</h1>
  <p>HTML is the content, CSS styles it, JS makes it dynamic.</p>
  <button id="btn">Click</button>
  <script>
    document.getElementById("btn").onclick = () => {
      alert("CSS + JS working with HTML!");
    };
  </script>
</body>
</html>

Practice Task

  • Create lesson2-demo.html.
  • Add a heading and a paragraph (HTML).
  • Use CSS to change background and text colors.
  • Add a button and show an alert with JavaScript.

Quick Quiz

  1. Which layer is responsible for structure?
  2. Which handles colors and layout?
  3. Which enables interactivity?
  4. Why separate HTML, CSS, and JavaScript?
Show sample answers
  • Structure: HTML
  • Presentation: CSS
  • Behavior: JavaScript
  • Separation: improves clarity, reuse, teamwork, and maintainability.
