Lesson 2 ‚Äî HTML vs CSS vs JavaScript

Learning Goals

  • Understand the role of each layer: HTML, CSS, JavaScript.
  • Explain why we separate structure, style, and behavior.
  • Recognize basic syntax differences.

1 ‚Äî HTML: Structure

HTML provides the skeleton of a webpage. It tells the browser what content exists.

<h1>My Website</h1>
<p>Welcome to my site.</p>
<a href="about.html">About me</a>
<img src="photo.jpg" alt="A photo">

2 ‚Äî CSS: Style

CSS describes how things look: layout, colors, fonts, spacing.

h1 {
  color: cyan;
  font-size: 2rem;
}
p {
  color: #b6b6b6;
}
img {
  border-radius: 8px;
}

3 ‚Äî JavaScript: Behavior

JavaScript controls interactivity ‚Äî reacting to user actions, changing the page, or fetching data.

<button id="clickMe">Click me</button>

<script>
  document.getElementById("clickMe")
    .addEventListener("click", () => {
      alert("You clicked the button!");
    });
</script>

Putting It All Together

A full page usually combines all three:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Demo</title>
  <style>
    body { background:#121212; color:#eee; }
    h1 { color: cyan; }
  </style>
</head>
<body>
  <h1>Hello!</h1>
  <p>HTML is the content, CSS styles it, JS makes it dynamic.</p>
  <button id="btn">Click</button>
  <script>
    document.getElementById("btn").onclick = () => {
      alert("CSS + JS working with HTML!");
    };
  </script>
</body>
</html>

Practice Task

  1. Create a new page lesson2-demo.html.
  2. Add a heading and a paragraph (HTML).
  3. Use CSS to change the background color and text color.
  4. Add a button, and use JavaScript to show an alert when clicked.

Quick Quiz

  1. Which layer is responsible for structure?
  2. Which handles colors and layout?
  3. Which enables interactivity?
  4. Why do we separate HTML, CSS, and JavaScript?
